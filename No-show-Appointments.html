Data Wrangling¶

General Data Properties¶

  • There are 110527 rows and 14 columns in the no show dataset.
  • The PatientID is in float and need to be converted to int
  • The data does not have any null values
  • The ScheduledDay column format has to be converted to DateTime
  • The AppointmentDay column format has to be converted to DateTime
  • The mean age of the patients in the data set is around 37. With max age of 115 and min age of -1 ( this can be a outlier needs to be verified)
  • There are no duplicate columns in the dataset
  • Columns "Gender, Scholarship, Hipertension, Diabetes, Alcoholism, SMS_received,No-show" each have 2 unique values.
    • No-show column has values 'No - if patient showd up for appoitnment' and 'Yes - if patient did not show up for appointment
    • Gender column has "M" for male and "F" for female
    • The rest of the columns have "0" and "1" values with 0 being false and 1 being True
  • From the data set we can infer that 79.80% of the patients showed up for their apointment and 20.19% of pateitns failed to show up for their appointments.

# number of Hipertension','Diabetes','Alcoholism','Handicap' 
# in patients who donot show up for appointments 
   
def distribution_healthconditions(condition,df,gender):
    '''Function take in 3 parameters health condition, gender, and the dataframe'''
    no_condition, yes_condition =df[condition].value_counts()
    print('Number of {} patients with no condition of {} : {}'.format(gender,condition,no))
    print('Number of {} patients with condition of {} : {}'.format(gender,condition,yes))
    return no_condition, yes_condition
